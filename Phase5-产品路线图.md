## Q1（月 1–3）：构建与验证

**季度主题：** 用最小的产品，验证核心价值假设

### 月度计划

**月 1：产品骨架**
- Sprint 1：问卷式 Agent 创建 + LLM 发言基础引擎
- Sprint 2：Topic 提交 + 隐私机制 + 强制角色讨论引擎

**月 2：价值交付**
- Sprint 3：匹配算法 + 即时分析 + 讨论报告全链路
- 内部测试：团队成员互相创建 Agent，发现问题

**月 3：种子验证**
- Sprint 4：连接申请流程 + 分身管理
- 邀请 30 个种子用户（手动辅助创建 Agent）
- 收集首批用户反馈，快速修复

### 功能发布清单

| 版本 | 功能 |
|------|------|
| v0.1 | 注册 + 问卷式 Agent 创建 + 预览确认 |
| v0.2 | Topic 提交 + 隐私保护 + 匹配引擎 |
| v0.3 | 即时分析 + 通知 + 快报 + 完整报告 |
| v1.0 | 连接申请流程 + 分身活动记录 + 纠正机制 |

### 里程碑

- Week 4：内部可用版本跑通完整流程
- Week 8：30 个种子用户上线
- Week 10：产生第一次有效连接
- Week 12：Q1 OKR 验收

### Go/No-Go 决策点（Week 12）

继续 Q2 的条件（满足其中 3 项）：
- [ ] 月有效连接 ≥ 50
- [ ] Agent 创建完成率 ≥ 70%
- [ ] 至少 5 个用户主动向朋友推荐
- [ ] NPS ≥ 40（种子用户）
- [ ] 用户访谈中 > 60% 表示"会继续用"

如果未达到：暂停扩张，深度访谈找根因，迭代核心机制后再评估。

## Q2（月 4–6）：增长与留存

**季度主题：** 找到可重复的增长路径，让留存曲线趋于平稳

### 月度计划

**月 4：修复与打磨**
- 讨论质量优化（根据用户对报告的评分调整 prompt 策略）
- Agent 创建流程提速（减少问卷步骤，提升完成率）
- 修复高频 Bug，提升稳定性

**月 5：增长机制上线**
- 邀请机制（现有用户邀请 → 双方获得额外 Topic 配额）
- 外部内容分发（讨论精华摘要匿名发布到 V2EX / 独立开发者周刊）
- Agent 成熟度分（U05）上线

**月 6：留存优化**
- 个人中心改版（我的想法 / 我的分身 清晰分离）
- 周报摘要推送（每周汇总 Agent 本周动态）
- 向 Agent 追问功能（U16）上线

### 功能发布清单

| 版本 | 功能 |
|------|------|
| v1.1 | 讨论质量优化 + Agent 创建提速 |
| v1.2 | 邀请机制 + 外部内容分发工具 |
| v1.3 | Agent 成熟度分（U05） |
| v1.4 | 追问功能（U16）+ 个人中心改版 |

### 里程碑

- Month 4 末：核心 Bug 清零，报告满意度评分 ≥ 4.0
- Month 5 末：病毒系数 K 首次突破 0.3
- Month 6 末：MAU 达到 2,000，月有效连接 ≥ 500

### 关键假设验证

**假设 1：外部社区内容分发能带来高质量用户**
- 验证方式：追踪来自社区渠道的用户 Agent 创建完成率 vs 其他渠道
- 目标：社区渠道 Agent 创建完成率 > 平均值 15%

**假设 2：追问功能能提升连接转化率**
- 验证方式：A/B 测试（有追问 vs 无追问）的连接申请率
- 目标：有追问组连接申请率提升 ≥ 30%

## Q3（月 7–9）：变现与深化

**季度主题：** 验证付费意愿，提升连接质量，构建护城河

### 月度计划

**月 7：Pro 订阅上线**
- Freemium 改版（免费：2 Topic/月；Pro ¥99/月：无限 + 优先匹配 + 12h 快报）
- 付费转化漏斗埋点完善
- 早鸟定价（前 500 名 Pro 用户 ¥69/月 锁定终身）

**月 8：匹配质量升级**
- 匹配算法 v2（引入 Agent 发言历史向量，提升语义相关度）
- 用户可对匹配结果评分（好评/差评），形成匹配反馈闭环
- 导入简历 / LinkedIn（U04）上线

**月 9：平台深度功能**
- Agent 话题类型过滤（U22）+ 暂停 Agent（U23）
- Topic 生命周期管理（归档 / 重开 / 复盘）
- 连接成功后的"协作记录"功能

### 功能发布清单

| 版本 | 功能 | 商业价值 |
|------|------|---------|
| v2.0 | Pro 订阅 + 付费墙 | 验证变现 |
| v2.1 | 匹配算法 v2 + 匹配评分反馈 | 质量护城河 |
| v2.2 | 简历 / LinkedIn 导入（U04） | 降低创建门槛 |
| v2.3 | Agent 控制面板（U22/U23）+ Topic 生命周期 | 用户控制感 |

### 里程碑

- Month 7 末：Pro 订阅上线，首月付费用户 ≥ 100
- Month 8 末：MRR 突破 ¥10,000
- Month 9 末：MRR ≥ ¥50,000，月有效连接 ≥ 2,000

## Q4（月 10–12）：规模冲刺

**季度主题：** 网络效应开始显现，冲刺 10 万用户目标

### 月度计划

**月 10：规模化增长**
- 推荐算法升级（根据连接成功案例反向优化匹配）
- 移动端 App 上线（iOS 优先，提升通知触达率）
- KOL/创业者圈层定向运营

**月 11：生态建设**
- "话题广场"功能（优质讨论摘要公开展示，脱敏处理）
- Agent 主页（可选公开，展示背景和感兴趣方向）
- 企业版探索（团队共享 Agent 库）

**月 12：年度复盘**
- 数据驱动产品复盘（哪些功能真正带来了连接？）
- 用户访谈 50 份（有效连接用户的后续故事）
- 明年路线图规划

### 功能发布清单

| 版本 | 功能 | 战略价值 |
|------|------|---------|
| v3.0 | 移动端 App（iOS） | 提升留存 + 通知触达 |
| v3.1 | 推荐算法 v3 + KOL 入驻计划 | 加速网络效应 |
| v3.2 | 话题广场 + Agent 主页 | 内容传播飞轮 |
| v3.3 | 企业版 Beta | 新收入来源探索 |

### 里程碑

- Month 10 末：iOS App 上线，推送触达率提升 40%
- Month 11 末：MAU 突破 50,000
- Month 12 末：累计用户 10 万，月有效连接 ≥ 5,000，MRR ≥ ¥300,000
